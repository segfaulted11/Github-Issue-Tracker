1️⃣ var vs let vs const:
var is function-scoped and can be redeclared, let is block-scoped and can change value, const is block-scoped but its value cannot be reassigned.

2️⃣ Spread operator (...):
... expands elements of an array or object, often used to copy or merge them.

3️⃣ map() vs filter() vs forEach():
map() creates a new array by transforming items, filter() creates a new array with items that pass a condition, forEach() just loops through items without returning a new array.

4️⃣ Arrow function:
A shorter way to write functions in JavaScript using =>.

5️⃣ Template literals:
Strings written with backticks ` ` that allow variable embedding using ${} and multi-line text.
